WSL 重启指南
====================

已为您创建 WSL 配置文件：C:\Users\Administrator\.wslconfig

此配置启用了镜像网络模式，可以让 WSL 共享 Windows 的 VPN 连接。

重启 WSL 步骤：
----------------

1. 在 Windows PowerShell 或 CMD 中执行：
   wsl --shutdown

2. 等待 5-10 秒

3. 重新打开 WSL 终端

4. 测试连接：
   cd /home/huyifan/projects/godzilla-evan/ref
   git clone http://172.16.12.71/gaowang/hf-open-live-demo.git

如果镜像网络模式不工作，请查看下面的备用方案。
